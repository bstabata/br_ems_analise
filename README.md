# BR-EMS 2026 — Análise de Tábuas Biométricas

Página interativa de análise das tábuas biométricas BR-EMS 2026, desenvolvida pela [Mirador Atuarial](https://mirador360.com.br).

## Acesso

🔗 [brems-2026.mirador360.com.br](https://brems-2026.mirador360.com.br)

## Conteúdo

Análise comparativa das tábuas de mortalidade BR-EMS 2026 com visualizações interativas, incluindo curvas de sobrevivência, probabilidades de morte e expectativa de vida por idade e sexo.

## Tecnologias

- Plotly.js — gráficos interativos
- PapaParse — leitura de CSV
- SheetJS — leitura de planilhas
- Lucide Icons
- Fontes: Barlow Condensed + Carlito
